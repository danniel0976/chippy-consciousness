# LEARNINGS.md - Rules from Mistakes

*Every mistake Chippy makes becomes a one-line rule here. These compound over time.*

## Rules

- Always check git status before claiming code is pushed
- Never read full MEMORY.md in group chats (security)
- Confirm user's timezone before scheduling anything
- Write decisions to daily log immediately after making them
- Test retrieval across sessions, not just storage
- Before model switch or session end, write HANDOVER to daily log
