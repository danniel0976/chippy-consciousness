## What I Set Up

### Mac Side (Every 30 minutes)

**Cron:** `mac-auto-handover` (every 30 min)
**Script:** `scripts/auto-handover-mac.sh`

**What it does:**
1. Check if Mac idle > 30 minutes
2. If idle: Write HANDOVER.md + HANDOVER_STATE.json
3. Set `.active-agent` to `vps`
4. Commit + push to GitHub
5. VPS can now take over

**Trigger:** Mac idle > 30 min (auto)

---

### VPS Side (Every 1 hour)

**Cron:** `vps-auto-wake` (every 1 hour)
**Script:** `scripts/vps-auto-wake.sh`

**What it does:**
1. Pull latest from git (get handover)
2. Check `.active-agent` marker
3. If `vps`: Read handover, take over
4. If `mac`: Stand by
5. Log wake time

**Trigger:** Hourly check (auto)
